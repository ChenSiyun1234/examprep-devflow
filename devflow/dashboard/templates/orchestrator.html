Review Queue (read-only orchestrator)

Shows the same cross-PR plan as orchestrate-reviews: who to request review from, findings to fix, mergeable / force-mergeable / ready-then-merge PRs, conflicts and retargets, and Codex rate-limit state. Advisory only — it recommends next actions and never mutates GitHub (no comments, reviewer requests, merges, mark-ready, retargets, pushes, or branch deletes), and it does not replace human approval. This is not a merge UI.

$notice $result

Read-only. Requires gh installed and authenticated. For each request review PR you get copyable @codex review text — the dashboard never posts it. For mergeable PRs it shows a human merge-preflight note, not a merge button.

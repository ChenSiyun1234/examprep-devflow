GPT fallback review prompt (read-only)

Builds a copyable prompt for a manual GPT/ChatGPT code review of a PR, from read-only GitHub data — useful when Codex is quota-limited or you want a second opinion. This page does not call GPT and sends your code nowhere; you copy the prompt and paste it yourself. For private/proprietary repos, use your own judgment before pasting into external tools.

$notice $result

Read-only; requires gh authenticated. The generated prompt instructs GPT to use a strict P1/P2/P3 findings format and to never suggest merge / push / branch-delete / GitHub Actions / secrets. Truncated diffs and feedback are flagged in the prompt.

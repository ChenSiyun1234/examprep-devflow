Codex review watcher (read-only)

Runs the same read-only sweep as watch-codex-reviews: scans open PRs for new trusted-Codex reviews and prints a marker. It only reads GitHub via gh and updates a local seen-file — it never comments, merges, or mutates anything.

$notice $result

Markers: ACTIONABLE_CODEX_REVIEWS > CODEX_WATCH_INCOMPLETE > CODEX_QUOTA_LIMITED > NO_NEW_CODEX_REVIEWS. Requires gh to be installed and authenticated.
